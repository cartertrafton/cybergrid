# cybergrid

Microgrid security simulation software.

Designed for WIT 2020 Computer Engineering Senior Design Project. 


## Program Structure
- ``main.py`` - Main program to run
- ``gui.py`` - User Interface built with Tkinter
- ``openPMU.py``
